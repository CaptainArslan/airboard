# Keyboard Shortcuts

## Tools

| Key | Action |
|-----|--------|
| 1 | Freehand |
| 2 | Line |
| 3 | Rectangle |
| 4 | Circle |
| 5 | Arrow |
| T | Text |
| E | Eraser (tool; use open palm to erase) |
| X | Clear canvas |

## Eraser

| Key | Action |
|-----|--------|
| E | Eraser tool (open palm also erases) |
| [ | Decrease eraser size |
| ] | Increase eraser size |
| + / - | Adjust eraser size when eraser tool is active |

Default eraser size: 30px (range 10–100px).

## Edit

| Key | Action |
|-----|--------|
| Z | Undo |
| Y | Redo |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| S | Save PNG to `exports/` |

## Brush

| Key | Action |
|-----|--------|
| + | Increase brush size |
| - | Decrease brush size |

## Window

| Key | Action |
|-----|--------|
| F | Toggle fullscreen |
| H | Toggle help |
| Q | Quit |
| Esc | Cancel text input, or quit |

## Text Tool

| Key | Action |
|-----|--------|
| T | Select text tool / open input |
| Enter | Commit text |
| Esc | Cancel text input |
